📦 SDK — TypeScript Client

Integrate InfraX into your app with one import

⚡

Install

npm install @infrax/sdk
TypeScript-first · Tree-shakeable · 0 deps
🛠

Usage

import { InfraX } from '@infrax/sdk';

const infrax = new InfraX({
  baseUrl: 'https://api.infrax.io',
  apiKey:  process.env.INFRAX_KEY
});

// Wallet, Vault, MPC, DC, Payment — all via one client
const wallets = await infrax.wallet.list();
const tx     = await infrax.vault.propose({...});
const events = await infrax.dc.subscribe({...});
🔗

API Reference
